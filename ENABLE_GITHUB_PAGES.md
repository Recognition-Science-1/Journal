# 🚀 Enable GitHub Pages for Recognition Science Journal

## Quick Setup (5 minutes)

### Step 1: Enable GitHub Pages
1. Go to your repository: https://github.com/Recognition-Science-1/Journal
2. Click **Settings** (top menu)
3. Scroll down to **Pages** (left sidebar)
4. Under **Source**, select **"Deploy from a branch"**
5. Choose **"main"** branch
6. Click **Save**

### Step 2: Wait for Deployment
- GitHub will automatically build and deploy your site
- This takes 2-5 minutes for the first deployment
- You'll see a green checkmark when it's ready

### Step 3: Access Your Live Website
Your Recognition Science Journal will be live at:
**https://recognition-science-1.github.io/Journal/**

## 🔧 Customization Options

### **Custom Domain (Optional)**
1. Buy a domain like `recognitionscience.org`
2. In GitHub Pages settings, add your custom domain
3. Enable HTTPS (recommended)

### **Theme Customization**
- Edit `_config.yml` to change theme
- Modify CSS in individual pages
- Add custom layouts in `_layouts/` folder

### **Analytics (Optional)**
Add Google Analytics by editing `_config.yml`:
```yaml
google_analytics: YOUR_TRACKING_ID
```

## 🚨 Troubleshooting

### **Site Not Loading?**
- Check GitHub Actions tab for build errors
- Ensure `_config.yml` syntax is correct
- Wait 5-10 minutes for propagation

### **Styling Issues?**
- Clear browser cache
- Check that CSS is properly formatted
- Verify Jekyll theme compatibility

### **404 Errors?**
- Ensure all internal links use `/Journal/` prefix
- Check that file names match exactly
- Verify permalink settings in frontmatter

## 🔗 API Access

Access the ledger programmatically:

```bash
# Get all truth packets
curl https://api.recognitionscience.org/v1/ledger

# Get specific packet
curl https://api.recognitionscience.org/v1/ledger/packets/{hash}

# Real-time updates via WebSocket
wscat -c wss://api.recognitionscience.org/v1/ledger/live
```
